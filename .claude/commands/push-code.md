---
description: "执行 git add、commit 并 push 到远程仓库"
---

请帮我将当前的代码更改推送到远程仓库。请按以下步骤操作：

1. 审查我的当前代码改动。
2. 根据这些改动，生成一个遵循 Conventional Commits 规范的提交信息。请用中文总结。
3. 将所有改动添加到暂存区（git add .）。
4. 使用上一步生成的信息进行提交（提交标题用 Conventional Commits；提交正文补充中文解说）。
5. 将当前分支推送到远程仓库（git push origin HEAD）。

执行前请先向我展示计划的提交信息，确认无误后再继续。
